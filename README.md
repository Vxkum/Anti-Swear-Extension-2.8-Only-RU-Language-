Anti-Swear 2.8 — это браузерное расширение, предназначенное для фильтрации русской ненормативной лексики.

Возможности:
-Замазывает до ~90% матерных слов и их производных.
-Работает автоматически на всех сайтах,не изменяя структуры страницы.
-Подходит для чтения текстов без раздражающей лексики.

Функции :
-Есть возможность включать и выключать.
-Есть возможность изменить тему виждета расширения на светлую/тёмную.
-Есть возможность менять способ замазывания текста: текст либо просто размазываеться / либо закрашивается в один из 6 предложенных цветов.
-В самом низу виджета расширения можно посмотреть работает расширение или нет.

Расширение будет полезно тем, кто хочет читать контент в более комфортной и нейтральной форме.
